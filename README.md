# DESAFIO-QA-BEEDOO-2026

Qual você acredita ser o objetivo da aplicação?
- O objetivo da aplicação é permitir o cadastro de novos cursos e os usuários verem os que já estão cadastrados.
  
Quais são os principais fluxos disponíveis?
1) Poder cadastrar cursos
2) Ver lista de cursos já cadastrados
3) Excluir cursos já existentes
4) Validar informações de cadastro

Quais pontos do sistema você considera mais críticos para teste?
1) Cadastro de informações e novos cursos
2) Verificar a lista de cursos para que estejam sendo exibidos e funcionando corretamente
3) Mensagem caso ocorra algum erro
